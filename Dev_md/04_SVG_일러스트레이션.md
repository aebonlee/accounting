## 네이밍 규칙
```
{토픽ID}-hero.svg          → 토픽 히어로 일러스트
{토픽ID}-{섹션ID}.svg      → 섹션 일러스트
```

## 사용 위치
- Hero SVG: 각 학습 페이지 상단 Hero 영역
- Section SVG: 각 섹션 컨텐츠 내 일러스트 (`<img src={...} alt={...} />`)
