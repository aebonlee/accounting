## 핵심 파일 설명

### 설정 파일
| 파일 | 역할 |
|------|------|
| `package.json` | 프로젝트 메타 (name: dreamitbiz-accounting) |
| `vite.config.ts` | Vite 빌드 설정, 개발 서버 포트 5186 |
| `.env` | Supabase URL/Key, VITE_SITE_URL |
| `index.html` | SEO 메타 태그, OG 태그 |
| `tsconfig.json` | TypeScript 설정 |

### 핵심 소스
| 파일 | 역할 |
|------|------|
| `src/config/site.ts` | 사이트 설정 + 8개 학습 경로 데이터 |
| `src/utils/translations.ts` | 한/영 번역 문자열 |
| `src/contexts/ThemeContext.tsx` | 테마 관리 (다크/라이트/자동, 5가지 컬러) |
| `src/contexts/LanguageContext.tsx` | 언어 전환 (ko/en) |
| `src/contexts/AuthContext.tsx` | Supabase 인증 상태 |
| `src/layouts/PublicLayout.tsx` | 라우트 + 레이아웃 구성 |

### 학습 페이지 패턴
각 학습 페이지(예: `Basics.tsx`)는 동일한 패턴을 따름:
1. `SECTIONS` 배열 — 6개 섹션 정의 (id, titleKo, titleEn, icon)
2. 사이드바 네비게이션 — 섹션 간 이동
3. Hero 영역 — SVG 일러스트 + 제목 + 설명
4. 섹션 컨텐츠 — 각 섹션별 한글/영어 학습 내용
5. 이전/다음 네비게이션 — 섹션 간 순차 이동
